# 🏥 MediChain Vault  
### *Your Health, Your Data, Your Control*

MediChain Vault is a **secure, blockchain-powered healthcare data management system** that gives patients full control over their medical records while enabling seamless and permission-based access for doctors.

---

## 🚀 Overview

Traditional Electronic Health Record (EHR) systems are:
- ❌ Centralized  
- ❌ Vulnerable to tampering  
- ❌ Limited patient control  

**MediChain solves this using Web3 + Encryption + Blockchain.**

---

## 🧠 Core Idea

Store **encrypted medical records off-chain** and keep **hash proofs on blockchain** for integrity and trust.

## 🔄 System Flow

### 👤 Patient Upload Flow
1. Patient uploads medical file  
2. File is encrypted (AES encryption)  
3. Encrypted file sent to backend  
4. Stored in secure vault  
5. SHA256 hash generated  
6. Hash stored on blockchain  

---

### 👨‍⚕️ Doctor Access Flow
1. Doctor requests access  
2. Backend checks permission  
3. Patient approves/rejects  
4. Encrypted file retrieved  
5. Doctor decrypts file  
6. Access logged (immutable)  

---

## 🔐 Security Features

- 🔒 AES-256 File Encryption  
- 🔗 SHA256 Hash Verification  
- ⛓ Blockchain Storage (Polygon)

## 🧪 How It Works

- File → 🔐 Encrypted  
- Encrypted File → ☁ Stored (Supabase)  
- File → 🔑 Hash Generated  
- Hash → ⛓ Stored on Blockchain  

✔ Any tampering = Hash mismatch → detected instantly
